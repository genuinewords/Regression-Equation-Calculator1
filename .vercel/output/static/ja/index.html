回帰方程式とは？

回帰方程式は変数間の関係をモデル化し、入力データに基づいて結果を予測できるようにします。最も単純な形式——単回帰——では、方程式はy = mx + bとなり、mは傾き、bはy切片です。この強力な統計ツールは、科学、ビジネス、工学でトレンドの発見、予測の作成、変数間の関係の定量化に使用されています。

当計算機を使う理由は？

  • データから回帰方程式を即座に計算
  • ステップバイステップの計算過程を表示し、数学を理解できる
  • R²、相関係数、傾き、切片を計算
  • 登録不要 — 100%無料でブラウザ上で動作
  • データはプライベートに保たれる — サーバーに一切送信しない

一般的な使用例

  • 広告費から売上を予測
  • 平方フィートから住宅価格を推定
  • 学習時間が試験成績に与える影響を分析
  • 時間経過による気温変化を予測
  • 投与量と反応の関係を評価

よくある質問

回帰方程式とは何ですか？
回帰方程式とは、従属変数と1つ以上の独立変数との関係を記述する数学的な公式です。単回帰では、y = mx + b の形をとります。ここで、m は傾き（変化率）、b はy切片（x = 0のときの値）です。この方程式により、独立変数の既知の値に基づいて従属変数の値を予測することができます。
回帰方程式の計算方法は？
線形回帰方程式を計算するには：1) xとyのデータの平均を計算する、2) 傾きを公式 m = Σ(xi - x̄)(yi - ȳ) / Σ(xi - x̄)² で計算する、3) 切片を b = ȳ - m·x̄ で計算する。当計算機はこれらすべての手順を自動で実行し、詳細な計算過程を表示します。
R二乗とは何ですか？また、どう解釈すればよいですか？
決定係数（R²）は、従属変数の分散のうち独立変数によって説明できる割合を表す統計量です。0から1の範囲をとります：R² = 0はモデルが変動性を全く説明していないことを意味し、R² = 1は完全な適合を意味します。一般的に、R²が0.7以上は強い関係、0.3～0.7は中程度、0.3未満は弱い関係を示します。
相関係数とは何ですか？
相関係数（r）は、2つの変数間の線形関係の強さと方向を測る指標です。-1から+1の範囲をとります：r = +1は完全な正の関係、r = -1は完全な負の関係、r = 0は線形関係がないことを意味します。rが±1に近いほど、線形の関連性は強くなります。
傾きと切片の違いは何ですか？
回帰方程式 y = mx + b において、傾き（m）はxが1単位増加したときにyがどれだけ変化するかを示します。正の傾きはxの増加に伴いyが増加することを、負の傾きはyが減少することを意味します。切片（b）はxがゼロのときのyの予測値であり、回帰直線がy軸と交わる点です。
回帰分析には何個のデータポイントが必要ですか？
単回帰方程式を計算するには最低2個のデータポイントが必要です。ただし、有意で信頼できる結果を得るには、10～15個以上のデータポイントを推奨します。データポイントが多いほど、回帰モデルの信頼性と統計的有意性が高くなります。
回帰の標準誤差とは何ですか？
回帰の標準誤差（SE）は、観測されたデータポイントが回帰直線から外れる平均距離を測ります。SEが小さいほどデータポイントが当てはめ直線に近いことを意味し、モデルの予測がより正確です。本質的に、モデルの精度を測る指標です。
いつ線形回帰を使うべきですか？
連続する従属変数と1つ以上の独立変数の関係をモデル化したいとき、またその関係がほぼ線形であると思われるときに線形回帰を使用します。一般的な応用例には、広告費から売上を予測する、床面積から住宅価格を推定する、学習時間がテストの点数に与える影響を分析するなどがあります。
線形回帰の前提条件は何ですか？
線形回帰は以下を前提とします：1) 線形性—変数間の関係が線形であること、2) 独立性—観測値が互いに独立していること、3) 等分散性—残差が一定の分散を持つこと、4) 正規性—残差が正規分布に従うこと、5) 多重共線性がない—独立変数間の高い相関がないこと（重回帰の場合）。これらの前提に違反すると、信頼できない結果につながる可能性があります。
相関と回帰の違いは何ですか？
相関は2つの変数間の関係の強さと方向を（-1から+1の範囲で）測りますが、因果関係を暗示しません。回帰はさらに進んで、データに直線を当てはめ、一方の変数から他方を予測できる方程式（y = mx + b）を提供します。相関は対称的（r_xy = r_yx）ですが、回帰は非対称です—xからyを予測するのとyからxを予測するので結果が異なります。
回帰にはどのような種類がありますか？
一般的な種類には以下が含まれます：線形回帰（1つの予測変数）、重回帰（複数の予測変数）、多項式回帰（曲線関係）、ロジスティック回帰（二値結果）、リッジ回帰（過学習を防ぐための正則化付き）、ラッソ回帰（正則化による特徴選択）。それぞれの種類は異なるデータパターンや研究質問に適しています。
