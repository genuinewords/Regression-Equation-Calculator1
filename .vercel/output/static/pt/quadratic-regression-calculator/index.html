Como calcular a regressão quadrática

A regressão quadrática ajusta uma parábola aos seus dados usando a equação y = ax² + bx + c. Esta é uma forma de regressão polinomial (grau 2) que captura relações curvas, em forma de U ou U invertido que uma linha reta não consegue modelar. Os coeficientes são encontrados resolvendo as equações normais — um sistema de três equações lineares construído a partir de somas de potências de x e produtos cruzados com y.

As fórmulas da regressão quadrática

Os coeficientes a, b, c satisfazem as equações normais:
nc + bΣx + aΣx² = Σy
cΣx + bΣx² + aΣx³ = Σxy
cΣx² + bΣx³ + aΣx⁴ = Σx²y
Resolvemos esse sistema 3×3 por eliminação gaussiana para obter a, b e c.

Exemplo de regressão quadrática

Suponha que seus dados sigam uma tendência parabólica: (−2,4), (−1,1), (0,0), (1,1), (2,4). Isso é exatamente y = x². A calculadora encontra a = 1.0, b = 0.0, c = 0.0, dando a equação y = 1.0x² + 0.0x + 0.0 com R² = 1.0 (ajuste perfeito). Dados reais não serão tão limpos, mas a regressão quadrática se destaca sempre que o gráfico de dispersão mostra uma curva clara.

Quando usar regressão quadrática

  • Um gráfico de dispersão mostra um padrão claro em forma de U ou U invertido
  • A regressão linear apresenta um R² ruim e você percebe curvatura sistemática nos resíduos
  • O fenômeno tem um pico ou vale natural (ex. desempenho vs. ativação, eficácia do fármaco vs. dose)
  • Você precisa de um modelo mais flexível, mas quer permanecer na família polinomial

Quando evitar regressão quadrática

  • A relação é aproximadamente linear — use nossa Calculadora de Regressão Linear
  • Você precisa de mais de um preditor — use nossa Calculadora de Regressão Múltipla
  • Curvatura de ordem superior (cúbica ou além) é necessária — considere regressão polinomial de grau 3+
  • Extrapolando muito além do intervalo dos seus dados — parábolas podem divergir rapidamente
