ما هو معامل ارتباط بيرسون؟

يقيس معامل ارتباط بيرسون (r) قوة واتجاه العلاقة الخطية بين متغيرين مستمرين. يتراوح من −1 (ارتباط سلبي تام) إلى +1 (ارتباط إيجابي تام)، حيث يشير 0 إلى عدم وجود علاقة خطية.

كيف تفسير r لبيرسون

  • |r| ≥ 0.9 — ارتباط قوي جداً
  • 0.7 ≤ |r| < 0.9 — ارتباط قوي
  • 0.5 ≤ |r| < 0.7 — ارتباط متوسط
  • 0.3 ≤ |r| < 0.5 — ارتباط ضعيف
  • |r| < 0.3 — ارتباط ضعيف جداً أو منعدم

متى تستخدم ارتباط بيرسون

  • كلا المتغيرين مستمران وتقريباً طبيعيان
  • العلاقة تبدو خطية
  • تريد قياس الارتباط وليس التنبؤ
  • لا توجد قيم شاذة متطرفة

متى تتجنب ارتباط بيرسون

  • العلاقة غير خطية (استخدم سبيرمان)
  • البيانات تحتوي على قيم شاذة
  • المتغيرات ترتيبية أو فئوية
  • تحتاج لإثبات السببية
